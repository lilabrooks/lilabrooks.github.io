Current themes

Four threads in the work.

They overlap: the same choices about boundaries, feedback, and ownership show up in platform teams and small software tools.

01

Agent work people can inspect

Using goals, behavior specs, architecture decisions, and verification commands to keep AI-assisted development tied to explicit intent.

02

Small local-first services

Building tools that solve one recurring problem, keep their data close, and remain understandable without a large runtime stack.

03

Platform operating models

Working through platform roadmaps, service intake, team charters, ownership, and cloud migration decisions as one connected system.

04

Applied AI inside real workflows

Following the gap between a good model demo and a production tool with clear boundaries, evaluation, observability, and human judgment.
